# 🎧 AdMuter – Spotify Ad Muting App

AdMuter is an Android app built with **Kotlin + Jetpack Compose + mvvm ** that automatically mutes audio when Spotify ads are playing, restores the volume once the music resumes, and tracks your listening stats,
counting ads and songs played.

---

## ✨ Features
- 🚫 Automatically **mutes Spotify ads**
- 🔊 Restores **volume when music resumes**
- 🧾 **Tracks number of ads**
- 🎶 **Tracks number of songs played**
- 📱 Runs in the background with a **notification listener**
- ⚡ Lightweight and battery-friendly


---

## 🛠️ Tech Stack
- **Kotlin**
- **Jetpack Compose (Material 3)**
- **NotificationListenerService**
- **Coroutines & Flow**
- **ViewModel (MVVM)**
